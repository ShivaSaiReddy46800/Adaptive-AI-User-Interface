# Adaptive-AI-User-Interface
Adaptive AI User Interface Project

This project demonstrates a simple Adaptive User Interface using Python and Tkinter.

The system learns user preferences such as theme (Dark Mode / Light Mode) and font size. 
When the user runs the program again, the interface automatically adapts based on previous settings.

Technologies Used:
Python
Tkinter
JSON

How to Run the Program:
1. Open the project in PyCharm
2. Run the file main.py

Project Files:
main.py
user_preferences.json
